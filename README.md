# CMP_INAR
This repo will do Bayesian exploration with CMP and INAR.
Attached is the Stan code. The results from the ML implementation using the R package COMPoissonReg and from Stan are very close (you can run the file COM-Poisson-reg-ToRun.R to verify this).\\
We have implemented the ZINAR(1) using CMP.
In the scripts, we generate from GP just to check if the CMP. stan code is running!
We also keep the stan code for ZINAR(1) using GP.
